Solución

Cada theme tiene su vuelta, su especialidad.

  1. Creá un backup

  2. Plugin> Add New

  3. Buscá por all in one wp migration > install now > activate

  4. All-in-One WP Migration > export > export to > file > Download
    Si la página pesa más de 300 megas es enorme y probablemente hizo algo mal.

  5. Andá al cPanel buscá por wordpress, dale click, y dale a Install

  6. Creá otra instalación de wordpress en otro directorio (new) para montar el backup o hacelo localmente en tu PC.

  7. En la nueva instalación de wordpress (new) instalá el plugin All-in-one WP Migration > activate Si la página es más grande que 300 megas, tenés que usar una versión paga de All-in-one WP migratoin (podés bajartela pirateada de por ahí: unlimited all-in-one wp migration-extension.zip

  8. Andate a la parte de plugins y dale a Upload plugin > choose file > install now > activate now

  9. All-in-one wp migration > import Si abajo dice: Maximum upload file size: 8Megabytes o alguna limitación en vez de decir Unlimited

  10. Andá al cPanel buscá PHP: select php version > options > upload_max_filesize > 1G memory_limit > 1G

  11. Ahora buscá por modsecurity > status > off (apagalo)

  12. Importá tu theme en All-in-one WP Migration y dale a proceed , y volvé a activar el modsecurity

  13. Ahora buscá php > select php version > options > y volvé lo que cambiaste: memory_limit > 512 y upload_max_filesize > 512 (o los valores que estaban)

  14. Settings > permalinks > logueate en tu pagina nueva > guardalo dos veces.

  15. Dashboard > Apareance > Themes > Add theme > blocksy > activate > install blocksy companion > skip

  16. El theme nos quitó el footer, el shop se ve diferente.

  17. Header > logo arrastralo al centro > select the logo > select

  18. Desactivá el site title y cambiá el logo height

  19. Click en menu1, select menu: Boverstir Menu, mové el menú abajo al centro y borramos el search

  20. Como los colores no son iguales, apretá menu, pestaña design, cambia la fuente a raleway: MEDIUM (mas chico)

  21. Quitamos el margen, apretá el logo de la página > diseño > reducí el margen lo dejó en -30(top), vacio(right), -47(bottom), vacio(left)

  22. Header > headers > global headers > sticky functionality > on

  23. Apretá el último

  24. Subí el logo arriba de todo

  25. Y el menú más arriba.

  26. Apretá en “main row” cambiá el tamaño de la fila.

  27. Publish

  28. Customizer > colores > global color palette, cambiá el color principal al que detectaste con la herramienta de f12 (dev tools) del logo.

  29. Podés poner de 2do color el color de los links que tienen al pasar el mouse por encima, lo podés pasar a imágen y detectar el color con gimp o con f12.

  30. Customize > general > layout > maximum size width > 1024 px

  31. Publish

  32. Edit with elementor (tenés que haberlo instalado)

  33. Apretá símbolo hamburguesa, user preferences > dark > editing handles.

  34. Atrás site settings > layout > content > 1024px > update

  35. Para resolver el ancho de el banner apretá arriba a la izquierda icono y editialo con elementor > advanced > padding > 0 0 0 0

  36. Ahora apretá el ícono que está a la derecha el lapiz > advanced > margin > 50 50 50 50. Padding > 50 50 50 150
