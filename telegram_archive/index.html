Necesito una aplicación, extensión, programa u otra cosa que archive todos los mensajes de varios canales de Telegram de los que soy miembro. Este canal desactiva copiar, guardar y reenviar mensajes. El programa debería poder retroceder hasta la fecha de inicio (aproximadamente el 16 de junio de 2024) y capturar capturas de pantalla de todos los mensajes hasta la fecha actual. Idealmente, el programa también transcribiría los mensajes a un formato de texto simple que mostrara el nombre de usuario asociado con el mensaje.

The program should be able to scroll back to the start date (approximately June 16, 2024) and screenshot every message up to the current date. Ideally, the program would also transcribe the messages to a simple text format that showed the username associated with the message.
