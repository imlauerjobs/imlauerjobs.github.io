We are seeking a skilled Data Analyst for a short-term, 3-hour project focused on performing data analysis using SQL. This is a fast-paced assignment, ideal for a candidate with experience in data extraction, manipulation, and analysis through SQL queries. You will be provided with a dataset, and your main objective will be to generate insights by running SQL queries and providing a summary report of your findings.

Key Responsibilities:

Use SQL to extract, filter, and analyze data from provided datasets. Perform data cleaning and transformation to ensure accuracy in analysis. Identify key trends, patterns, and anomalies within the dataset. Generate a summary report, outlining key insights and actionable findings. Deliver concise and clear documentation of the SQL queries used.

Requirements:

Proficiency in SQL: Ability to write complex queries involving joins, aggregations, subqueries, and data transformations. Experience working with relational databases. Strong attention to detail and the ability to quickly identify trends and insights. Problem-solving skills: Ability to interpret results and present actionable recommendations. Must be able to complete the task within 3 hours.

Deliverables:

SQL queries used for data extraction and analysis. A short summary report of key insights and recommendations based on the data analysis. Documentation of any assumptions or considerations made during the analysis. This project is perfect for a detail-oriented individual with the ability to work independently and under time constraints. If you can deliver high-quality work within the required time, we’d love to hear from you!
