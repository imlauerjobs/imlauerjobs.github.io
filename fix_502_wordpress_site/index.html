Causas del problema

  1. Mucho tráfico (DDOS)
  2. navegadores viejos
  3. problemas de seguridad de firewalls

Rapidas soluciones

  1. Refrescá la página.
  2. Limpiá la cache y las cookies (o probá con modo incógnito)
  3. A veces tener dos plugins que hagan la misma función puede causar problemas en el servidor esto es muy común con los plugins de cache:
en el administrador de wordpress andá a plugins > installed plugins
fijate si tenés dos plugins de caché instalados, si es así, desactivá uno de ellos.
Si esto no funciona desactivá todos los plugins temporalmente y fijate si funciona.

Si este error no te deja ingresar a tu panel de control (dashboard) vas a tener que hacerlo desde el cpanel.

  1. Fijate la configuración de la DNS: si moviste tu página a un nuevo hosting esto cambia a donde los dns apuntan, esto sucede cuando la ip de tu pagina cambia. Si este es la fuente del problema tenés que esperar 24 horas para que el DNS se propague.

  2. Deshabilita tu CDN.
